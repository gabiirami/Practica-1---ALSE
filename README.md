# Practica 1 - ALSE

Este repositorio contiene la solución a los ejercicios solicitados en la práctica.

##  Organización

- **MCD/**
  - Cálculo del Máximo Común Divisor usando el algoritmo de Euclides.

- **InteresCompuesto/**
  - Programa que calcula el monto final aplicando interés compuesto.

- **Arreglos/**
  - Suma de los elementos de un arreglo usando un ciclo `for`.

- **LeetCode/**
  - `palindrome.cpp` → Determina si un número es palíndromo.
  - `two_sum.cpp` → Encuentra dos elementos cuya suma es un valor objetivo.

##  Objetivo

Aplicar lógica de programación y estructuras básicas en C++ para resolver problemas algorítmicos.
